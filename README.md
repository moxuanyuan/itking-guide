##JavaScript
- **必知必会** 
  - 原生类型、内置类型。 
  - 基本类型的相关函数。 
  - 对象类型的判定方式。 
  - prototype的作用、类型创建、继承。 
  - ECMA基本语法，如分号插入规则、逗号运算符、括号的多义性等。
